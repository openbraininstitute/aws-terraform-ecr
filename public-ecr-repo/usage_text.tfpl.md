## Usage

You can run these container images in any container based environment, for example with Podman or Docker on your laptop, but also within Kubernetes or on cloud providers such as AWS.

Please check the Dockerfile in the [GitHub repository](${github_repo}) for a list of expected environment variables.

Please note that this container is part of the neuroscience platform provided by the [Open Brain Institute](https://www.openbraininstitute.org/) which is not that straightforward to configure from scratch.
If you just want to experiment with neuroscience, then it might be easier to create a free account on the [Open Brain Institute](https://www.openbraininstitute.org/) platform instead.
